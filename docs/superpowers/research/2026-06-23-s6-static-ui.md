# S6 — Public UI (static, no-build)

**Date:** 2026-06-23
**Where:** `integritybg/site/` — open `index.html` directly (file://; no server, no `npm install`).
**Design system:** ui-ux-pro-max → *Data-Dense Dashboard* (blue `#1E40AF` + amber accent, green/amber/red
risk status, Fira Sans / Fira Code with tabular figures, WCAG AA, no emoji icons, 150–300ms transitions).

## Why static
Disk is full → no room for a React `node_modules`. The site is plain HTML/CSS/JS; `export_site.py`
bakes the SQLite into `data.js` (`window.DATA` + `window.GEO`, 460 KB) so it runs by double-click with
no `fetch`/CORS. Leaflet loads from CDN; charts are pure CSS (no chart lib).

## Pages
- **Карта (`index.html`)** — Leaflet **choropleth of the 28 области** colored by average risk
  (green <0.45 / amber 0.45–0.60 / red ≥0.60 / grey = no mapped data). Hover highlight + tooltip;
  click an област → side panel (contracts, value, single-bid %, high-risk count, avg risk). KPI strip on
  top. Region is matched from buyer name → **22% honest geo-coverage**, stated in the UI.
- **Анализи (`analytics.html`)** — KPI strip + a flag-summary bar chart + **six sortable tables**:
  top-risk contracts (with risk badge + indicator chips), изпълнители, възложители/общини, сектори (CPV),
  and the shared-owner network. Click any column header to sort (keyboard-accessible, `aria-sort`).

## Honesty & a11y baked in
- Every page leads with a **disclaimer**: automatic risk *signals* by objective indicators — not
  accusations or proof. Risk badges use **dot + text label** (never colour alone).
- Sortable headers are keyboard-operable; tabular figures; row-hover; focus-visible rings;
  `prefers-reduced-motion` respected; Cyrillic throughout.

## Build / refresh
```bash
python export_site.py     # DB -> site/data.js (+ inlined geojson)
# open integritybg/site/index.html
```
